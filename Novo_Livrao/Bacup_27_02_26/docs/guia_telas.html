Tela de Login

💻 LoginForm
📁 src/features/Auth/LoginForm.jsx  •  view = 'login'

O que faz

Permite ao usuário entrar no sistema com e-mail e senha. É a primeira tela que o usuário vê ao acessar o app.

Elementos visíveis

  • Logo "LIVRÃO DA FAMÍLIA"
  • Campo de E-mail
  • Campo de Senha (com botão mostrar/ocultar)
  • Botão "Entrar"
  • Link "Ir para Minha Conta"
💡 Como se referir
Diga: "Na tela de Login"
🎉

Tela de Boas-Vindas

💻 WelcomeScreen
📁 src/features/Welcome/WelcomeScreen.jsx  •  view = 'welcome'
🎉 Tela de transição — aparece brevemente
após o login antes de ir ao Console.

O que faz

Exibida logo após o login. Dá as boas-vindas ao usuário pelo nome antes de direcioná-lo ao painel principal (Console).

💡 Como se referir
Diga: "Na tela de boas-vindas"
🏠

Console / Painel Principal

💻 UserHub
📁 src/features/Hub/UserHub.jsx  •  view = 'hub'

O que faz

É o menu central do app. A partir daqui, o usuário escolhe para onde ir: Árvore, Álbum, Receitas ou Configurações.

Elementos visíveis

  • Nome do usuário (ex: "David Israel")
  • Palavra "CONSOLE" abaixo do nome
  • Card "Árvore Familiar"
  • Card "Álbum Digital"
  • Card "Receitas"
  • Botão "Minha Conta (Configurações)"
  • Link "Sair com segurança"
💡 Como se referir
Diga: "No Console" ou "No Painel Principal" ou "No Hub"
🌳

Árvore Familiar

💻 FamilyTreeSelector
📁 src/components/FamilyTreeSelector.jsx  •  view = 'form', passo 2

O que faz

Mostra a árvore genealógica da família com cards de cada membro, organizados hierarquicamente. Permite selecionar um familiar para editar.

Elementos visíveis

  • Barra de progresso "PROGRESSO TOTAL DA FAMÍLIA X%"
  • Abas: Identidade, Família e vida, Trajetória, Memórias Vivas
  • Cards: Bisavós → Avós → Pais → Representante → Irmãos
  • Barras de progresso coloridas em cada card
  • Botão "+ PARENTE" para adicionar membros
  • Botão "SAIR" para voltar ao Console
💡 Como se referir
Diga: "Na Árvore Familiar" ou "Na tela da Árvore"
❌ Evite: "repositório da árvore"
📋

Formulário de Edição

💻 FormEngine
📁 src/features/Form/FormEngine.jsx  •  view = 'form', passo ≥ 3

O que faz

Formulário para preencher/editar os dados de um familiar. Possui várias seções organizadas em abas.

Seções (Abas)

Aba Nome Técnico Conteúdo
Identidade Identity Nome, apelido, nascimento
Família e Vida FamilyVital Conjugal, filhos, vitais
Trajetória LifeCulture Educação, profissão
Memórias Vivas Narrative Histórias, relatos

Botões do Header

  • 🟧 Sair sem Salvar
  • 🟥 Apagar Parente
  • 🟦 Salvar Dados
💡 Como se referir
Diga: "No formulário de edição" ou "Na ficha do Pai"
📸

Galeria de Fotos

💻 PhotoGalleryModal
📁 src/features/Gallery/PhotoGalleryModal.jsx
📸 Modal (janela flutuante) — abre sobre
a tela atual para ver/adicionar fotos.

O que faz

Janela flutuante (modal) que abre sobre a tela atual. Permite ver e adicionar fotos de um familiar específico.

Como acessar

  • Pelo card "Álbum Digital" no Console
  • Pelo ícone de câmera na NavBar do Formulário
💡 Como se referir
Diga: "Na galeria de fotos" ou "No álbum do [Nome]"
🍳

Receitas da Família

💻 RecipesPage
📁 src/features/Recipes/RecipesPage.jsx  •  view = 'recipes'

O que faz

Página para preservar receitas tradicionais da família. Cada receita pode ter ingredientes, modo de preparo, tags e informações de contexto.

Elementos visíveis

  • Header "Receitas de Família"
  • Botão "Nova Receita"
  • Cards de receitas (ex: "Gefilte Fish da Vovó")
  • Tempo de preparo, rendimento
  • Tags: Shabat, Médio, etc.
💡 Como se referir
Diga: "Na página de receitas" ou "No livro de receitas"
⚙️

Minha Conta

💻 AccountSettings
📁 src/features/Account/AccountSettings.jsx  •  view = 'account'

O que faz

Permite ao representante atualizar seus dados pessoais: foto, telefone, cidade e profissão.

Elementos visíveis

  • Foto do usuário (Selfie)
  • Botão "Atualizar Selfie"
  • Telefone (WhatsApp)
  • E-mail de login (somente leitura)
  • Cidade Atual
  • Profissão
💡 Como se referir
Diga: "Na tela de Minha Conta" ou "Nas configurações"
✅

Tela de Sucesso

💻 view = 'success'
📁 Componente inline em App.jsx
✅ Aparece após salvar dados com sucesso.
"Cadastro Salvo com Sucesso!"

O que faz

Aparece após salvar dados com sucesso. Confirma ao usuário que tudo foi gravado.

Elementos visíveis

  • Ícone de check verde
  • "Cadastro Salvo com Sucesso!"
  • Botão "Incluir ou Editar Membro Familiar"
  • Botão "Voltar ao Console"
💡 Como se referir
Diga: "Na tela de confirmação" ou "Na tela de sucesso"
